box-sizing 요소의 크기 계산법

  • box-sizing:content-box;
    width: 300px; height: 100px;
    기본값. width와 height의 값에
    padding이나 border의 값이 포함되지 않음.
    (추가시 레이아웃에 변동이 있을 수 있음)
  • box-sizing:border-box;
    width: 300px; height: 100px;
    width와 height의 값에
    padding이나 border의 값이 포함됨.
    (추가시 레이아웃에 변동이 없음)

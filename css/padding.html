padding - 테두리 기준, 내부의 여백

padding의 방향별 지정 속성

  • padding-top: 20px;
    테두리 기준 윗쪽 여백을 지정
  • padding-right: 50px;
    테두리 기준 오른쪽 여백을 지정
  • padding-bottom: 1em;
    테두리 기준 아랫쪽 여백을 지정
  • padding-left: 20%;
    테두리 기준 왼쪽 여백을 지정

padding의 값 갯수별 지정법

  • padding:n ; 양사방이 동일한 값으로 적용됨
  • padding:n n; 상하와 좌우가 서로 동일한 값으로 적용됨
  • padding:n n n; 상 좌우 서로 동일 하 순서로 적용됨
  • padding:n n n n; 상 우 하 좌 순서로 적용됨

position - 요소의 위치 지정법

  • position:static; 기본값. 위치 이동 불가
  • position:relative; 현재 위치를 기준으로 이동
  • position:absolute; 부모요소를 기준으로 이동
  • position:fixed; 브라우저를 기준으로 고정
  • position:sticky; relative와 fixed의 믹스

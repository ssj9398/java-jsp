transition

  • transition-property : ; 효과를 줄 속성명
  • transition-duration : ; 효과를 줄 시간 (단위는 s나 ms)
  • transition-timing-function : ; 가속 감속 등
    1. ease; 기본값. 천천히 시작해서 느리게 끝남
    2. ease-in; 천천히 시작
    3. ease-out; 천천히 끝남
    4. ease-in-out; 시작과 끝 모두 천천히
    5. linear; 등속
    6. cubic-bezier(n,n,n,n); 베지에 곡선값을 이용
      cubic-bezier.com
  • transition-delay : ; (효과를 주기 전)지연 시간
  • transition: ; 함축형 표현. 속성값만 나열

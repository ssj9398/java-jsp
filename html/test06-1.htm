html5에서 추가된 input type들

  • time : 시분초 지정가능한 상자
  • datetime-local : 년월일시분초 지정가능한 상자
  • date : 년월일 지정가능한 상자
  • week : 년주 지정가능한 상자
  • month : 년월 지정가능한 상자
  • number : 숫자값을 지정가능한 상자
    1. min="n" 속성으로 최소값을 지정
    2. max="n" 속성으로 최대값을 지정
    3. step="n" 속성으로 증감 간격을 지정
  • range : 범위내의 값을 지정가능한 상자
  • tel : 전화번호 형식을 지정가능한 상자(가상키패드 변화)
  • email : 이메일형식을 지정가능한 상자(가상키패드 변화)
    브라우저가 알맞는 형식인지 체크해줌(@ 여부..)
  • url : url주소형식을 지정가능한 상자(가상키패드 변화)
    브라우저가 알맞는 형식인지 체크해줌(프로토콜 여부)
  • search : 검색필드 지정가능한 상자(삭제버튼)
  • color : 색상값을 지정가능한 상자

html5에서 추가된 input type의 속성들

  • placeholder=abc : 입력예시를 보여줌
  • required(required="required") : 필수 입력요소로 지정
  • autofocus : 해당 요소에 커서가 미리 위치
  • autocomplete="on/off" : 자동완성 기능을 켬/끔
  • readonly : 입력영역이 읽기전용으로(변경불가)
  • disabled : 입력영역이 비활성화

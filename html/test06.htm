복습이 필요한 과목은?

input type=hidden - 방문자에게는 보이지 않는 데이터 전송용

input type=file - 파일 전송용

input type=submit/reset/button - (전송/초기화/일반) 버튼 생성

value 속성의 값으로 버튼의 글자를 변경 가능

input type=image -전송역할의 버튼을 대체하는 이미지를 지정

src속성으로 이미지 경로를 지정

button 버튼의 글자 /button

button태그는 반드시 종료태그와 함께 쓰며 글자를 그 사이에 기입함.
기본적으로 전송역할을 하는데
취소역할을 하려면 type="reset",
일반 버튼은 type="button"로 지정하면 됨.
다른 태그를 내부에 쓸 수 있어 다양한 활용 가능.

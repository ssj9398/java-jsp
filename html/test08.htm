문제. 아이티윌부산 로고이미지를 표시하고 클릭하면 사이트로 이동하는 링크 걸기 (새 탭에 열리며 마우스오버시 설명 글자에 '아이티윌부산으로 이동' 문구를 표시하기)

target - 링크를 여는 방식 지정

  1. target="_self" - 현재 창 열기 (기본값)
  2. target="_blank" - 새 탭에 열기
  3. target="_top"- 프레임구조 없이 브라우저 전체에 하나로 열기
  4. target="_parent" - 한 단계 상위 창(부모창)에 열기
